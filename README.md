# Tic-Tac-Toe Game

## Overview

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game allows two players to take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner.

## Features

- Two-player game
- Interactive game board
- Display of the current player's turn
- Announcement of the winner or a draw
- Reset button to start a new game

## Technologies Used

- HTML
- CSS
- JavaScript
